# CellarApp

This folder contains the generated-project path for the first dedicated iOS app target.

## Files
- `project.yml` — XcodeGen spec
- `CellarApp/` — app sources and assets
- `CellarAppUITests/` — UI smoke tests

## Generate project

```bash
../../scripts/dev/generate-ios-project.sh
```

## Run simulator smoke test

```bash
../../scripts/dev/test-ios-simulator.sh
```
